# Scala People
A Scala application to manage people in an organization (just for learning Scala)

## Setup

```
$ git clone git@github.com:dmatthew/scala_people.git
$ cd scala_people
$ vagrant up
```

## Running the application

```
$ vagrant ssh web
$ cd /vagrant/play_test/scala-people
$ sbt run
```

## Tech Stack

* Scala
* sbt
* MongoDB
